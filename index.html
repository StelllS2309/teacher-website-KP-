Настройки email-отправки
Каждое КП отправляется с отдельного email-адреса.
Для каждого аккаунта Mail.ru / BK.ru:
1. Войдите → Настройки → Безопасность → Пароли для внешних приложений → Создать
2. Название: «КП», скопируйте сгенерированный пароль
3. Вставьте его в поле «Пароль» ниже (основной пароль не подойдёт)
1. Зайдите на resend.com → «Sign in with GitHub»
2. Добавьте домен: Domains → Add Domain
3. Создайте API-ключ: API Keys → Create (начинается с re_)
1. Зарегистрируйтесь на mailjet.com
